# Fonctions du plugin

Les fonctions du plugin RevitAddin sont organisées en panneaux dans l'onglet **ndc26** du ruban Revit.

| Panneau | Fonctions |
| --- | --- |
| [Vues](vues/) | Créer Vue Longitudinale, Coter Quadrillage, Zone de Coupe, Plage de Vue |
| [Nuage de point](nuage-de-points/) | Afficher/Masquer Nuage, Plancher Nuage |
| [Murs & Ouvertures](murs-et-ouvertures/) | Créer Type Mur, Créer Type Porte, Créer Type Fenêtre |
| [Etiquetage & Feuille](etiquetage-et-feuilles/) | Etiqueter Structure, Date Cartouche |
| [Outils](outils-de-suivi/) | Temps Projet, Stats Usage |
